# Position Collection Refactor — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Fix float-equality tier sibling bug, add closed-order guard, switch to O(1) UUID-keyed storage, split the 8-param trailing stop method into two focused methods, and replace all int timestamps with DateTimeImmutable.

**Architecture:** Option A (one SellOrder per tier) + Approach 2 (bug fixes + collection cleanup, no DDD layer boundary changes). A shared `positionGroupId: Uuid` is generated once per position entry in `SimulationEngine::sellOrdersFromExitPlan` and assigned to all tier siblings, replacing the float buyPrice equality lookup. The single `ratchetTrailingStops` method splits into `ratchetTrailingStopsPercent` and `ratchetTrailingStopsAtr`; mode dispatch moves to the callers.

**Tech Stack:** PHP 8.4, Symfony 7.3, Doctrine ORM, Codeception 5 (PHPUnit 10). All commands run inside the `paybis-app` Docker container. Tests: `composer test-unit` (unit suite), `composer test` (all). Style: `composer cs-fix`. Static analysis: `composer phpstan`.

public function findByGroupId(Uuid $groupId): array
{

public function moveTierSiblingsToBreakeven(SellOrder $exclude, Uuid $groupId): void

    }

### Step 3: Implement

Remove the old `ratchetTrailingStops` method from `OpenPositionCollection.php`. Add the private helper and two public methods:

```php
private function isActivated(
    SellOrder $order,
    int $activationCandles,
    \DateTimeImmutable $currentTime,
    int $intervalSeconds,
): bool {
    if ($activationCandles <= 0 || $intervalSeconds <= 0) {
        return true;
    }
    $heldSeconds = $currentTime->getTimestamp() - $order->getEntryTime()->getTimestamp();
    return ($heldSeconds / $intervalSeconds) >= $activationCandles;
}

public function ratchetTrailingStopsPercent(
    float $candleHigh,
    float $trailingPercent,
    int $activationCandles,
    \DateTimeImmutable $currentTime,
    int $intervalSeconds,
): void {
    foreach ($this->orders as $order) {
        if (!$this->isActivated($order, $activationCandles, $currentTime, $intervalSeconds)) {
            continue;
        }
        $newStop = $candleHigh - $candleHigh * ($trailingPercent / 100);
        if ($newStop > $order->getStopLossPrice()) {
            $order->updateStopLoss($newStop);
        }
    }
}

public function ratchetTrailingStopsAtr(
    float $candleHigh,
    float $atr,
    float $atrMultiplier,
    int $activationCandles,
    \DateTimeImmutable $currentTime,
    int $intervalSeconds,
): void {
    foreach ($this->orders as $order) {
        if (!$this->isActivated($order, $activationCandles, $currentTime, $intervalSeconds)) {
            continue;
        }
        $newStop = $candleHigh - ($atr * $atrMultiplier);
        if ($newStop > $order->getStopLossPrice()) {
            $order->updateStopLoss($newStop);
        }
    }
}
```
